# Built With

Cognivox was built using a modern, high-performance stack optimized for real-time multimodal AI interaction.

## 🧠 AI & Machine Learning
- **Gemini 2.5 Flash Live API**: The core multimodal reasoning engine for real-time voice and vision.
- **Imagen 3**: High-fidelity image generation for financial summary cards and infographics.
- **Veo 3.1**: Cinematic video generation for immersive financial journey visualizations.
- **Google AI SDK (`@google/genai`)**: The official SDK for low-latency WebSocket communication.

## 💻 Frontend
- **React 18**: Component-based UI architecture.
- **Vite**: Ultra-fast build tool and development server.
- **TypeScript**: Type-safe development for robust application logic.
- **Tailwind CSS**: Utility-first styling for a "Technical Dashboard" aesthetic.
- **Motion (Framer Motion)**: Fluid layout animations and page transitions.
- **Lucide React**: Clean, consistent SVG iconography.
- **Web Audio API**: Real-time PCM audio processing, encoding, and decoding.
- **MediaStream API**: Capturing synchronized camera and microphone input.

## ⚙️ Backend
- **Node.js 22**: High-performance JavaScript runtime with native TypeScript support.
- **Express.js 5.0**: Minimalist web framework for serving static assets and configuration APIs.
- **HTTP/WebSockets**: Bi-directional communication for real-time AI interaction.

## ☁️ Cloud & Infrastructure
- **Google Cloud Platform (GCP)**: The primary cloud provider for hosting and scaling.
- **Google Compute Engine (GCE)**: `e2-medium` virtual machine instances.
- **PM2**: Production process manager for Node.js applications.
- **Google Cloud VPC Firewall**: Secure network configuration for Port 3000 access.
- **Bash & Gcloud CLI**: Automated infrastructure-as-code deployment scripts.

## 🛠 Tools & DevOps
- **NPM**: Package management and dependency handling.
- **ESLint & Prettier**: Code quality and consistent formatting.
- **Rsync**: Efficient file synchronization between local development and cloud VMs.
- **Tar**: Compressed asset packaging for fast cloud transfers.
